# Python Beginner Projects

This repository contains beginner-level Python projects to practice core Python programming concepts such as:

- Variables, Data Types
- Loops and Conditional Statements
- Functions
- Basic Object-Oriented Programming (OOP)
- Simple Console Applications

---

## Projects Included

1. **Student Management System**
   - Console-based application to add, update, delete, and view student records.

2. **Calculator Application**
   - Simple calculator to perform addition, subtraction, multiplication, and division.

3. **Number Guessing Game**
   - Fun game using loops and random number generation.
